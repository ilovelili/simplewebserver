# Simple Web Server

simple web server writen in `GO`

## Spin up

```bash
docker-compose up
```

## Endpoints

| URL | Method | Note |
| --- | --- | --- |
| `/` | `GET` | return root |
| `/` | `POST` | return root |
| `/count` | `GET` | return counter |
| `/count` | `POST` | increase singleton counter |
